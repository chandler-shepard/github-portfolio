Wireframe Intro

Task:
The goal of this project was to theoretically develop a website for yourself. To do this I created wireframes of what I wanted the site to look like. From there I used them in user testing, testing what worked and what could be changed in the site. After that I used the feedback from my user testing to create an updated version of the wireframes.
