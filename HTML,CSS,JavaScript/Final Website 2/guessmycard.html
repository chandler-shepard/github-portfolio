About The Task:
The purpose of this task was to allow the user to guess a playng card rank and suit. From there the user would be told if they got the suit correct and if the rank that they chose was either too high or too low. This task was built in python using html and cgi. inorder to complete this I had to have a program that would

  • Use a form to get information from the user
  • Compare that information with a random generated rank and suit
  • Output the results into a new page
  • Use the guessed rank and suit to post the correct playing card they guessed
